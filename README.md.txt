 Student Management System (SQL)

## Description
Beginner-level Student Management System built using MySQL.
This project stores and manages student and course data.

## Tools Used
- MySQL
- MySQL Workbench

## Features
- Student records
- Course records
- Student enrollment
- SQL queries for data retrieval

## How to Use
1. Open MySQL Workbench
2. Create a new schema
3. Import student_management.sql
4. Execute the script
